# Contribuer au Cours Bash

Merci de l'intérêt que vous portez à ce cours ! Nous voulons rendre cette ressource aussi complète et précise que possible.

## Comment Contribuer

### Signaler un Bug
Si vous trouvez une erreur dans le code ou dans les explications, n'hésitez pas à ouvrir une "Issue".

1. Utilisez un titre clair et descriptif.
2. Décrivez le problème en détail.
3. Si possible, proposez une correction.

### Proposer une Amélioration
Vous avez une idée pour un nouveau chapitre, un exercice ou une meilleure explication ?

1. Forkez le dépôt.
2. Créez une branche pour votre fonctionnalité (`git checkout -b feature/ma-nouvelle-idee`).
3. Committez vos changements (`git commit -m 'Ajout: Explication sur X'`).
4. Pushez vers la branche (`git push origin feature/ma-nouvelle-idee`).
5. Ouvrez une Pull Request.

## Règles de Conduite
- Soyez respectueux et constructif dans vos échanges.
- Assurez-vous que votre code est lisible et commenté.
- Testez vos scripts avant de soumettre.

Merci pour votre aide !
